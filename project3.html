Project Overview

This project is a mobile application prototype for the No Arena Philadelphia campaign, inspired by my personal connection to Philadelphia’s Chinatown and my concern about the proposed stadium that could displace the community. Through this prototype, I support advocacy efforts by transforming a website without a mobile counterpart into a mobile-friendly mockup, making it easier for users to take action and stay informed.

The application focuses on key user tasks such as signing petitions, donating, learning about the cause, and viewing upcoming events. In my prototype, I emphasize two primary interactions: navigating from the homepage to explore upcoming events (including detailed pages for two events), and accessing and completing the petition signing process.
